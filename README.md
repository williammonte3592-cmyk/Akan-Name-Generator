# Akan Name Generator

## Description

The Akan Name Generator is an interactive web application that calculates the day of the week a user was born, then assigns them a traditional Akan soul name (Kra Din) from Ghanaian culture.
In the Akan tradition, every child is given a day name at birth that corresponds to the day of the week. This name carries spiritual meaning and is believed to reflect the character and destiny of the child's soul — a cherished practice passed down through generations as a powerful link to Ghanaian heritage.

## Setup Instructions

1. Clone this repository
2. Open index.html in your web browser
3. Enter your birth date and select your gender
4. Click submit to see your Akan name

## BDD (Behavior Driven Development)

1. Input: Birth date and gender

   - Output: Akan name based on day of birth and gender

2. Input: Invalid date (day ≤ 0 or day > 31)

   - Output: Error message requesting valid date

3. Input: Invalid month (month ≤ 0 or month > 12)
   - Output: Error message requesting valid month
